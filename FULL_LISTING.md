# Полный листинг проекта АИИО РБ

## Структура проекта

```
web_ais/
├── index.html                    # Главная страница
├── css/
│   └── styles.css               # Все стили (1162 строки)
├── js/
│   ├── config.js                # Конфигурация приложения
│   ├── supabase-client.js       # Клиент Supabase
│   ├── api.js                   # API функции (705 строк)
│   ├── auth.js                  # Аутентификация (380 строк)
│   ├── utils.js                 # Утилиты (597 строк)
│   ├── validators.js            # Валидаторы (272 строки)
│   ├── app.js                   # Основной файл приложения
│   └── pages/
│       ├── dashboard.js         # Дашборд (260 строк)
│       ├── institutions.js      # Учреждения (490 строк)
│       ├── students.js          # Учащиеся (455 строк)
│       ├── staff.js             # Работники (462 строки)
│       ├── statistics.js        # Статистика (356 строк)
│       ├── reports.js           # Отчеты (389 строк)
│       ├── import.js            # Импорт данных (379 строк)
│       ├── settings.js          # Настройки (239 строк)
│       └── admin.js             # Администрирование (331 строка)
├── supabase_schema.sql          # Схема базы данных (173 строки)
├── README.md                    # Документация проекта
└── .gitignore                   # Игнорируемые файлы
```

## Общая статистика

- **Всего файлов**: 16
- **Общий объем кода**: ~6,500 строк
- **Основные языки**: HTML, CSS, JavaScript
- **База данных**: Supabase (PostgreSQL)

## Основные компоненты

### 1. Конфигурация (config.js)
- Настройки Supabase
- Типы учреждений образования
- Роли пользователей
- Типы отчетов
- Категории статистики

### 2. API слой (api.js - 705 строк)
- CRUD операции для всех сущностей
- Работа с Supabase REST API
- Фильтрация, сортировка, пагинация
- Логирование действий

### 3. Аутентификация (auth.js - 380 строк)
- Вход/регистрация через Supabase Auth
- Управление сессиями
- Ролевая система доступа
- Смена пароля

### 4. Утилиты (utils.js - 597 строк)
- Форматирование дат и телефонов
- Уведомления и модальные окна
- Навигация по страницам
- Экспорт данных в CSV
- Пагинация

### 5. Валидаторы (validators.js - 272 строки)
- Валидация email, пароля, телефона
- Валидация форм учреждений, учащихся, работников
- Отображение ошибок валидации

### 6. Страницы приложения

#### Дашборд (dashboard.js - 260 строк)
- Статистические карточки
- График учреждений по типам
- Последняя активность
- Быстрые действия

#### Учреждения (institutions.js - 490 строк)
- Таблица с фильтрацией и сортировкой
- CRUD операции
- Просмотр деталей
- Пагинация

#### Учащиеся (students.js - 455 строк)
- Управление списком учащихся
- Привязка к учреждениям
- Фильтрация по учреждению и классу

#### Работники (staff.js - 462 строки)
- Управление персоналом
- Фильтрация по должности
- Привязка к учреждениям

#### Статистика (statistics.js - 356 строк)
- Графики по типам учреждений
- Графики по регионам
- Динамика по годам
- Экспорт в Excel

#### Отчеты (reports.js - 389 строк)
- Генерация отчетов в PDF, Excel, Word
- Быстрые шаблоны отчетов
- История созданных отчетов

#### Импорт (import.js - 379 строк)
- Импорт из Excel/CSV
- Предпросмотр данных
- Шаблоны для загрузки
- Статистика импорта

#### Настройки (settings.js - 239 строка)
- Управление профилем
- Смена пароля
- Настройки уведомлений
- Системные настройки (для админа)

#### Администрирование (admin.js - 331 строка)
- Управление пользователями
- Системные логи
- Резервное копирование
- Системная информация

## База данных (supabase_schema.sql - 173 строки)

### Таблицы
1. **profiles** - профили пользователей
2. **institutions** - учреждения образования
3. **students** - учащиеся
4. **staff** - работники
5. **statistics** - статистические данные
6. **reports** - отчеты
7. **logs** - логи действий
8. **notifications** - уведомления

### Политики безопасности (RLS)
- Row Level Security для всех таблиц
- Разграничение доступа по ролям
- Изоляция данных пользователей

### Индексы
- Оптимизация запросов по ключевым полям
- Индексы по внешним ключам
- Индексы по часто используемым полям

## Технологический стек
